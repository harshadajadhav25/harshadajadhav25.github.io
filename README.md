## 📊 Featured Projects

### 🤖 InsightAI – AI-Powered Data Analyst Agent

A Generative AI-powered data analytics application that enables users to upload CSV datasets and interact with their data using natural language.

InsightAI uses an LLM to interpret analytical questions, dynamically generate Pandas analysis logic, execute data analysis, explain results in natural language, and automatically generate relevant visualizations.

**Key Features:**
- Natural language data analysis
- Generative AI integration using Groq API
- Dynamic Pandas analysis generation
- Automated data cleaning and quality checks
- AI-powered data visualizations
- Dataset health monitoring
- Conversational context for follow-up questions
- Validation of AI-generated analysis code

**Tech Used:** Python, Generative AI, Groq API, Pandas, Streamlit, Plotly

🔗 **Live Demo:** https://ai-data-analyst-agent-csv.streamlit.app/  
🔗 **GitHub:** https://github.com/harshadajadhav25/ai-data-analyst-agent
